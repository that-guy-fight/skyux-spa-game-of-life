From the home screen:
  1. Enter a non-negative integer into the text input field.

  2. Press the enter key to create a grid that will have a number of rows and columns equal to the number that was input into the text input field.

  3. After the grid has been generated, all tiles are initially set to 'dead'. Clicking a dead cell will toggle it to 'alive'. Clicking on an alive cell will toggle it back to dead.

  4. Click any number of cells toggling them from alive or dead until you have a desired pattern established within the grid.

  5. Click the 'Activate' button to start the game. You must have entered a number in the text input field and pressed the enter key in order to enable the 'Activate' button.

  6. Clicking the 'Pause' button will pause the game in its current state. While the game is paused, you can click and toggle any of the cells in the grid.

  7. Clicking 'Activate' while paused will resume the game. Cells cannot be toggled while the game is running.

  8. Clicking the 'Reset' button at any time will return the game to its intial state.

  9. It is recommended to not zoom out within your browser window too much, as it will distort the appearance of the cells.
